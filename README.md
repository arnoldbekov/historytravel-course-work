# HistoryTravel

**HistoryTravel** — Интерактивная карта культурного и исторического наследия региона.Сайт создан в рамках курсовой работы как учебный проект. Информация носит ознакомительный характер и может содержать неточности.

Проект представляет собой веб-приложение, позволяющее пользователям изучать исторические объекты региона, просматривать информацию по эпохам, сравнивать фотографии прошлого и настоящего, а также планировать маршруты путешествий.

## Возможности сайта

**Интерактивная карта** с отметками объектов культурного и исторического наследия
**Исторические эпохи** — описание ключевых периодов истории региона
**Галерея** — сравнение фотографий «тогда и сейчас»
**Маршруты** — готовые тематические маршруты и возможность создания своих
**Избранное** — сохранение понравившихся объектов и маршрутов
**Адаптивный дизайн** — удобный просмотр на смартфонах, планшетах и компьютерах
**Страница «О проекте»** — описание цели, функционала и инструкция по использованию
